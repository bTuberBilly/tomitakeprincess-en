# Foreward

Thank you all for your interest in this translation of Tomitake Princess.

This is my second major translation project. And while it definitely started as a bit of a joke (hence the April Fools Day release), I'd be lying if I said I didn't have an absolute blast working on this.

This is the kind of work that you really only see once every few years, for better and *definitely* for worse. It truly is one of a kind, and I sincerely hope you enjoy your time on Tomitake Island.

# Installation

1) Download the original freeware game from [here](http://yakiniku-banzai.com/down/tp.exe)
2) Install the game
   1) As a side note, if you are not on a Japanese locale, the installer will probably look funny, but it's still fairly easy to navigate this way.
3) Download the latest version of the patch from the [releases tab](https://github.com/bTuberBilly/tomitakeprincess-en/releases)
4) Extract the contents of the patch into the root of the game folder
5) Enjoy your stay on Tomitake Island! 🌴
